# Ruan Bester - Portfolio

A premium personal portfolio website showcasing creative development, design, and digital craftsmanship.

## Features

- 🎨 Premium dark + gold aesthetic
- ✨ Smooth animations and interactive elements  
- 📱 Fully responsive design
- 🌍 Interactive globe showing location (South Africa)
- 📧 Contact form with validation
- 🎯 Sections for About, Skills, Work, Awards, and Experience

## Tech Stack

- HTML5
- CSS3 (custom, no frameworks)
- Vanilla JavaScript
- Cobe Globe library (CDN)

## Live Demo

https://bessicodes.github.io/portfolio

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required - it's pure HTML, CSS, and JS

## Structure

```
├── index.html      # Main HTML file
├── css/
│   └── style.css   # All styling
├── js/
│   └── main.js     # Interactive features
└── images/         # Portfolio images
```
